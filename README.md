# CTF Writeups Blog

Retro gaming-themed static blog for CTF writeups.

## Setup

```bash
npm install
npm run dev
```

## Build

```bash
npm run build
```

The `dist` folder will contain the static site ready for deployment.

## Features

- Fetches writeups automatically from GitHub repo
- Retro gaming UI with pixel fonts (Jacquard 12, Pixelify Sans)
- Markdown rendering for WRITEUP.md files
- Fully static React app
